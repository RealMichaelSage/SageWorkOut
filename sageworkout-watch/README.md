# SAGEWorkOut для Xiaomi Mi Band 9

Это мини-приложение для браслета на базе Zepp OS 3.0.

## Как запустить на часах

### 1. Подготовка окружения
На вашем Mac должен быть установлен **Node.js** (v16+).
Установите официальный инструмент разработчика Zepp:
```bash
npm install -g @zeppos/zeus-cli
```

### 2. Сборка проекта
Перейдите в эту директорию и установите зависимости (если появятся в будущем):
```bash
cd sageworkout-watch
zeus dev
```

### 3. Просмотр в симуляторе
Вы можете скачать **Zepp OS Simulator** с официального сайта [docs.zepp.com](https://docs.zepp.com/). После запуска симулятора выполните:
```bash
zeus preview
```

### 4. Установка на браслет
1. В приложении **Zepp** на телефоне включите "Режим разработчика" (Developer Mode).
2. Подключите телефон к той же Wi-Fi сети, что и ваш Mac.
3. Выполните команду:
```bash
zeus bridge
```
4. Отсканируйте появившийся QR-код через приложение Zepp.

## Особенности
- **Авто-таймер**: 60 секунд отдыха после каждого сета.
- **Вибрация**: Оповещение об окончании отдыха.
- **Безопасность**: Напоминания о технике прямо на экране упражнения.
